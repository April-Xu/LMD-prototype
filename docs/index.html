CSS/FILE Structure

Mainly three main CSS files are used in this theme. The first one bootstrap.css, second one is main.css and third one is style.css which for this template style.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Google Web Fonts

By default, the template loads this font from Google Web Font Services, you can change the font with the one that suits you best.

                                                            
                                                            @import url('https://fonts.googleapis.com/css?family=Montserrat:100,200,300,400,500,600,700,800,900');

Contact Form/ MailChimp Subscription

Contact

We have added an attractive and fully functional contact form. You need to update the recipient name and email address.

You can change Subject and email address in contact.php file in contact_form folder

    //change where to mail sent 
    $PHPMailer->AddAddress('info@pentathemes.com');
    $PHPMailer->Subject = "photomania Contact";
